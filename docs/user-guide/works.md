# 作品登録

## 概要

作品登録機能は、お気に入りの作品を記録し、独自の評価システムで管理できる機能です。アニメ、漫画、ゲーム、映画、小説など、様々なジャンルの作品を登録し、「絶対相対評価」システムに基づいて評価できます。

### 主要な特徴

- **多様なカテゴリ対応**: アニメ、漫画、ゲーム、映画、小説など
- **詳細な分類**: カテゴリごとのサブカテゴリで細かく分類
- **独自の評価システム**: ティア1〜7の7段階評価
- **作品情報の管理**: 作家名、初出年、規模、完結状況など
- **タグ機能**: 自由にタグを追加して整理

---

## 主要機能

### 1. カテゴリ選択

作品のメインカテゴリを選択します:

- **アニメ**: TVアニメ、劇場版アニメなど
- **漫画**: 少年漫画、少女漫画、青年漫画など
- **ゲーム**: コンシューマーゲーム、PCゲーム、モバイルゲームなど
- **映画**: 実写映画、アニメ映画など
- **小説**: ライトノベル、文芸書など

### 2. サブカテゴリ選択

選択したカテゴリに応じて、さらに詳細な分類が可能です:

**アニメの例**:
- 冒険
- 格闘
- 大河
- ラブコメ
- BL

**漫画の例**:
- 少年漫画
- 少女漫画
- 青年漫画
- 女性漫画
- ラブコメ
- スポーツ

### 3. 評価システム(ティア)

作品を7段階で評価できます:

| ティア | 評価内容 |
|--------|----------|
| **ティア1** | 最高評価。人生を変えるレベルの作品 |
| **ティア2** | 非常に優れた作品。強くおすすめ |
| **ティア3** | 良い作品。おすすめできる |
| **4: 普通または合わない** | 可もなく不可もなく、または自分に合わなかった |
| **5: 未評価** | まだ評価していない(デフォルト) |
| **6: 未読** | まだ読んでいない、見ていない |
| **7: おすすめ** | 他の人におすすめしたい作品 |

## 使い方

### ステップ1: カテゴリの選択

1. 「作品登録」ページにアクセス
2. **カテゴリ**のドロップダウンから、作品のジャンルを選択
   - 例: 「アニメ」を選択

### ステップ2: サブカテゴリの選択(任意)

1. カテゴリを選択すると、サブカテゴリが表示されます
2. 該当するサブカテゴリにチェックを入れます(複数選択可)
   - 例: 「ラブコメ」「大河」を選択

### ステップ3: 作品情報の入力

必須項目と任意項目を入力します:

#### 必須項目
- **作品名**: 作品のタイトルを入力
- **評価(ティア)**: 1〜7から選択(デフォルトは「5: 未評価」)

#### 任意項目
- **作家名**: 作者、監督、クリエイター名
- **初出年**: 作品が最初に発表された年
- **作品の規模**:
  - 小: 短時間で読み、見終わる
  - 中: 何時間もかかる
  - 大: 何日もかかる
- **適正年齢幅**: 対象年齢の範囲(スライダーで調整)
- **完結状況**: 完結している場合はチェック
- **発売国**: 作品が発売された国
- **言語**: 作品の言語
- **公式サイトURL**: 公式サイトのリンク
- **作品紹介**: 作品の簡単な説明や感想
- **作品検索URL**: 検索エンジンでの検索リンク
- **アフィリエイトURL**: 購入リンク(管理者のみ)

### ステップ4: タグの追加(任意)

1. 「タグを追加」フィールドにタグ名を入力
2. 「追加」ボタンをクリック
3. 複数のタグを追加して、作品を整理できます
   - 例: 「SF」「タイムリープ」「歴史物」

### ステップ5: 登録

すべての情報を入力したら、**「登録」ボタン**をクリックして保存します。

## よくある質問

### Q1: 作品を削除できますか?
**A**: はい、登録した作品は後から削除できます。作品一覧ページから該当作品を選択し、「削除」ボタンをクリックしてください。

### Q2: 評価を変更できますか?
**A**: はい、いつでも評価を変更できます。作品詳細ページから「編集」を選択し、ティアを変更してください。

### Q3: サブカテゴリは複数選択できますか?
**A**: はい、サブカテゴリは複数選択可能です。該当するすべてのサブカテゴリにチェックを入れてください。

### Q4: 「未読」と「未評価」の違いは?
**A**:
- **未読(ティア6)**: まだ作品を見ていない、読んでいない状態
- **未評価(ティア5)**: 作品を見たが、まだ評価をしていない状態

### Q5: 管理者向け機能とは?
**A**: 管理者は、公式作品情報の追加や、アフィリエイトURLの設定など、追加の機能を使用できます。一般ユーザーは基本的な作品登録と評価のみ可能です。

### Q6: タグは何個まで追加できますか?
**A**: タグの数に制限はありませんが、整理のしやすさを考えて、1作品あたり3〜5個程度を推奨します。

### Q7: 作品名のふりがなは必要ですか?
**A**: 作品名のふりがな(読み仮名)は任意項目です。検索の精度を上げたい場合は入力することをおすすめします。

## Tips

### 効率的な作品登録

1. **一括登録**: 複数の作品を続けて登録する場合、カテゴリが同じならそのまま次の作品を入力できます
2. **テンプレート活用**: 似たような作品(例: 同じシリーズ)は、前回の入力内容をコピーして効率化
3. **タグの再利用**: 一度入力したタグは、入力候補として表示されます

### おすすめの使い方

- **未読リスト作成**: 「ティア6: 未読」で作品を登録し、読みたい作品リストを作成
- **評価の見直し**: 時間が経ったら、作品を再評価して評価を更新
- **友人との共有**: リスト機能と組み合わせて、おすすめ作品を友人と共有
